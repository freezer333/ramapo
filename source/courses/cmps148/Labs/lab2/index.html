Lab 02 - Estimating ex with loops

The constant “e” is approximately 2.718 and has been calculated to 869,894,101 decimal places.

ex is approximated by the following series.

  ex = x0/0! + x1/1! + x2/2! + x3/3! + x4/4! +  … + xn/n!

Write a program that asks the user for a positive value for X and the number of terms (N) to use.  Display ex based on the above approximation using N terms.

e^5 (2 iterations) = 6.0000000000

e^5 (6 iterations) = 91.4166666667

e^5 (26 iterations) = 148.4131590981

e^5 (126 iterations) = 148.4131591026

Note - I used setprecision(10) and fixed to display full precision.
